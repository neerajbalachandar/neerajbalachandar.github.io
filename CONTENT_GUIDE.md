w# Content Guide

This guide shows the quickest way to add or edit content on the site.

## Add a research project

1. Create a new folder under `content/project/` with a short slug (example: `content/project/my-project/`).
2. Add an `index.md` in that folder using the same front matter as existing projects.
3. Add a thumbnail image named `featured.svg` (or `featured.jpg/png`) in the same folder.
4. Add tags `ongoing` or `past` to control where the project appears.

## Add a news item

1. Create a new folder under `content/post/` with a slug (example: `content/post/2026-my-update/`).
2. Add an `index.md` and include the tag `news` so it appears on the homepage.

## Add a publication or manuscript

1. Create a new folder under `content/publication/`.
2. Add an `index.md` and fill in the `title`, `summary`, and `publication` fields.
3. Use `publication: "Manuscript in preparation"` for drafts and update when published.

## Update your bio

Edit `content/authors/admin/_index.md` to update your bio, interests, and experience.
